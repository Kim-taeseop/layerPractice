#### Entity
- description 폴더 : Entity 관련 코드 및 설명을 모은 엔티티
    - cascade 폴더: 영속성 전이 및 고아 객체 설명을 위한 엔티티
    - inher 폴더: 상속관계 매핑을 위한 엔티티
    - mapping 폴더: 연관관계 매핑, 기본 엔티티 설명 포함
    - Period : 임베디드 타입 설명
- jpaEntity 폴더 : JPA에 필요한 엔티티 모음
- BaseEntity : @MappedSuperclass 설명 엔티티
